## 5. Metrics to Track

### Financial Metrics

| Metric | Target | Why It Matters |
|--------|--------|----------------|
| **CAC (Customer Acquisition Cost)** | <$200 | Must be low for self-serve model |
| **LTV (Lifetime Value)** | >$2,500 | 12x CAC minimum for healthy unit economics |
| **LTV:CAC Ratio** | >12:1 | Validates pricing + retention |
| **Churn (Monthly)** | <5% | Target: 2-3% for managed services |
| **NPS (Net Promoter Score)** | >50 | Target: 60+ for viral growth |

### Usage Metrics

| Metric | Target | Why It Matters |
|--------|--------|----------------|
| **Credits consumed/user/day** | 10-15 | Indicates active engagement |
| **Shadow Manager actions/day** | 5-10 | Shows proactive value delivery |
| **Health Score accuracy** | >85% | Predictive power validation |
| **Learning loop effectiveness** | 15% reduction in corrections/mo | Self-improvement proof |

### Tracking Implementation

```sql
-- Daily CAC/LTV tracking query
SELECT 
  date,
  SUM(marketing_spend) / COUNT(DISTINCT new_customers) as CAC,
  AVG(lifetime_value) as LTV,
  AVG(lifetime_value) / (SUM(marketing_spend) / COUNT(DISTINCT new_customers)) as LTV_CAC_ratio
FROM metrics_daily
GROUP BY date;
```
