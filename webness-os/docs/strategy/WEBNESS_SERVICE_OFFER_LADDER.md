# Webness Service Offer Ladder

Date: June 4, 2026

## Pricing Gate

Accessible premium. Webness should not chase low-ticket clients.

| Offer | Price |
| --- | ---: |
| Signal Scan | Rs. 9,999-Rs. 24,999 |
| Website / Brand System Sprint | From Rs. 75,000 |
| App / Dashboard / Portal Sprint | From Rs. 2.5L |
| Growth System Retainer | From Rs. 25,000/mo |
| Premium Systems Retainer | From Rs. 75,000/mo |

## Fit Rules

- If the buyer cannot pay for Signal Scan, they are not a good-fit build client.
- If the problem is unclear, sell diagnosis before build.
- If the work needs ongoing improvement, route to retainer.
- If the client wants isolated low-cost pages, decline or defer.

## Margin Rule

Every accepted offer should target 60%+ gross margin.
